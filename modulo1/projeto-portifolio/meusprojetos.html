Projeto Loja de Tênis Esportivos

Top Tênis Brasil

Esse projeto foi desenvolvido para um cliente que ainda não possuía sua loja física na web. Desse modo, o site foi pensado para melhor atenteder as demandas dos clientes e dos funcionários. Utilizou-se as linguagens HTML e CSS para a construção do site.

Além da Página Inicial, a loja conta com páginas exclusivas para: controle estoque, controle de funcionarios e adição de novos produtos.

A seguir, serão abordados algumas páginas e etapas do referido projeto:


Página de controle de Estoque:



Para a criação do controle de estoque, no que diz respeito aos elementos do HTML, foram utilizadas as TAGs que se criam as tabelas (table, th, td, tr). Outro ponto importante foi a inserção das imagens referentes ao podruto, para melhor compreensão, por meio da TAG img.

No que diz respeito ao CSS, destacou-se as bordas da tabela para um melhor efeito visual e também, a estilização das cores, conforme pensado pelo cliente.



Página de adição de prodrutos:


No que tange construção da Pagina de adicionar produtos, houve a nessidade de se trabalhar com formularios, dessa forma, através das TAGs: Form, Label, imput, button, etc; criou-se toda a estrututura para preenchimento de dados.

Para o CSS, a estilização permaneceu a mesma, dando uniformidade ao projeto como um todo, visto que todas as páginas interagem entre-si.
